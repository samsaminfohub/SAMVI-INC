# Scripts

Utility scripts for the IT Support Chatbot project.

## Available Scripts

### setup.sh
Initial project setup script for Unix/Linux/macOS.

```bash
chmod +x scripts/setup.sh
./scripts/setup.sh
```

### setup.bat
Initial project setup script for Windows.

```cmd
scripts\setup.bat
```

### rebuild_index.sh
Rebuild the FAISS vector index.

```bash
chmod +x scripts/rebuild_index.sh
./scripts/rebuild_index.sh
```

### backup.sh
Backup documents and index.

```bash
chmod +x scripts/backup.sh
./scripts/backup.sh
```

## Usage Guidelines

1. **Make scripts executable**: `chmod +x script_name.sh`
2. **Add documentation**: Comment your scripts
3. **Error handling**: Include proper error checks
4. **Logging**: Add logging for debugging
